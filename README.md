# AnimeRecSys - ИИ Рекомендации Аниме для AnimeLib

Браузерное расширение, которое интегрирует искусственный интеллект для персонализированных рекомендаций аниме непосредственно на сайте AnimeLib.

## Описание

Это расширение для браузеров на базе Chromium (Chrome, Edge, Opera) и Firefox, которое добавляет секцию с рекомендациями аниме на страницы просмотра аниме на сайте AnimeLib. Рекомендации генерируются с помощью ИИ на основе текущего просматриваемого аниме.

## Функции

- **Персонализированные рекомендации**: Получает рекомендации на основе аниме, которое вы сейчас просматриваете
- **Интеграция с сайтом**: Бесшовная интеграция с интерфейсом AnimeLib
- **Загрузочная анимация**: Приятная анимация во время генерации рекомендаций
- **Обработка ошибок**: Грамотная обработка ошибок и отображение сообщений пользователю

## Установка

### Установка из Chrome Web Store

Последняя стабильная версия расширения доступна в [Chrome Web Store](https://chromewebstore.google.com/detail/kkhankeacfldhbegodaofpibkngebjne?utm_source=item-share-cb). Просто перейдите по ссылке и нажмите "Установить".

### Ручная установка

#### Для Chrome/Edge/Opera
1. Скачайте архив с релизом из папки `Releases/` (например, `Ver 0.8.5.zip`)
2. Распакуйте архив
3. Откройте браузер и перейдите в `chrome://extensions/`
4. Включите "Режим разработчика" в правом верхнем углу
5. Нажмите "Загрузить распакованное расширение"
6. Выберите папку `release` из распакованного архива
7. Расширение установлено!

#### Для Firefox
1. Скачайте архив с релизом
2. Распакуйте архив
3. Откройте Firefox и перейдите в `about:debugging`
4. Нажмите "Этот Firefox" слева
5. Нажмите "Загрузить временное дополнение..."
6. Выберите файл `manifest.json` из папки `release`
7. Расширение установлено!

## Использование

1. Установите расширение
2. Перейдите на любую страницу аниме на сайте AnimeLib (anilib.me, animelib.org и т.д.)
3. Наслаждайтесь персонализированными рекомендациями в новой секции "Рекомендации"

## Разработка

### Структура проекта

```
Anime-Rec-Plugin/
├── dev/                    # Версия для разработки
│   ├── manifest.json       # Манифест с версией -dev
│   ├── background.js       # Service worker для API запросов
│   ├── content.js          # Content script для вставки рекомендаций
│   └── ai.gif/ai1.gif      # Анимация загрузки
├── release/                # Релизная версия
│   ├── manifest.json       # Манифест без -dev
│   ├── background.js
│   ├── content.js
│   └── ai.gif/ai1.gif
└── Releases/               # Архивы предыдущих релизов
    ├── Ver 0.8.3.zip
    └── Ver 0.8.5.zip
```

### Сборка релиза

1. Убедитесь, что код в `dev/` протестирован
2. Скопируйте файлы из `dev/` в `release/`
3. Удалите `version_name` из `release/manifest.json`
4. Создайте ZIP-архив папки `release/`
5. Переместите архив в папку `Releases/` с соответствующим именем версии

### API

Расширение использует внешний API для генерации рекомендаций:
- URL: `http://158.160.175.208:8000/api/get_recs_by_single_title`
- Метод: POST
- Payload: `{ "animeLibId": number, "top_k": number }`
